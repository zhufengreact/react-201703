#### Atom的使用
```
    Atom工具下载官网：https://atom.io
```    
###### --Atom使用方法--
```
     0. 设置atom 左则的导航样式背景颜色及字体大小   File->Settings->Open Config Folder->style.less->.tree-view{font-size:15px;}

     1. Ctrl+Shift+D            整行快速复制 代码块
     2. Ctrl+D                  快速创建中括号 {}  
     3. Ctrl+Shift+H            编辑器实时预览 HTML 页面   插件名:atom-html-preview
     4. 页面中间线去掉方法        修改 Preferred Line Length   修改值为：0
     5. Ctrl+\                  隐藏目录
     6. Alt+\                   显示目录
     7. Ctrl+/                  单行注释    
     8. Atom编辑器配置           http://imweb.io/topic/56c12f7e5c49f9d377ed8f1e
```
